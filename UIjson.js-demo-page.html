UIjson.js A Jquery plugin with bootstrap's responsiveness

-- View and Edit your Json models with a responsive bootstrap Interface --

Things to know

  • It is dependant from Jquery.js and Bootstrap and If you want ability to sort array Items with cursor you will need Jquery.ui also.
  • It is responsive cause of bootstrap's classes,expandable and with an organised code architecture according to Jquery PlugIn Authoring
  • Only properties of your model which have string,boolean,number or an array of these types as value, can be defined to UIjson and be rendered with bootstrap
  • Browser support : IE 9,IE 10,IE 11, Firefox, Safari, Opera , Chrome. UIjson.js follows browser support of Jquery.js v 2+ and Bootstrap
